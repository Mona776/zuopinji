EXPERIMENT_LOG

The Color Palette Generator was conceptualized as a response to static color pickers. I wanted to build a tool that felt alive—a digital organism that responds to noise patterns and user velocity to generate harmonious color combinations.

system_prompt > HYPOTHESIS "By mapping Perlin noise values to HSL color scales, we can generate palettes that feel organic yet maintain strict accessibility compliance for modern UI standards."

Throughout the development phase, I focused on the "Tactile Response" of the interface. Every click triggers a software-like mechanical sound and a sharp CSS transform that simulates physical depth.

Findings_01

Randomized seed generation resulted in 84% user satisfaction compared to manual selection.

Findings_02

Contrast ratio calculations are performed in real-time using a background worker.
